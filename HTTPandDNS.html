This page contains information on Hypertext Transfer Protocol and Domain Name System




What is HTTP


HTTP is Hyper Text Transfer Protocol. It is part of internet. It allow transmitting webpages, audio, images, videos, and other file types on internet between server and browser. Browser is operating system which send the request using the text GET, and server is operating system which receives the request from client and respond with the files requested. When the your browser request file using the text GET, HTTP request send to server. And when server get HTTP request, and server send the files which is requested by your browser through HTTP response. And when your browser get files, it display to your screen so that you can see.

What is DNS


DNS is Domain Name System
