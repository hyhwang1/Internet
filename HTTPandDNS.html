This page contains information on Hypertext Transfer Protocol and Domain Name System




What is HTTP


HTTP is HyperText Transfer Protocol. It is part of the internet. It allows transmitting webpages, audio, images, videos, and other file types on the internet between server and browser. Browser is the operating system which send the request using the text GET, and server is operating system which receives the request from the client and responds with the files requested. When your browser request file using the text GET, HTTP requests are sent to the server. And when the server gets an HTTP request, the server sends the files which are requested by your browser through HTTP response. And when your browser gets files, it displays to your screen so that you can see.

What is DNS


DNS is Domain Name System. It translate names like www.example.com into the numerical IP addresses (like 192.0.2.1) that computers use to locate each other on the internet.
